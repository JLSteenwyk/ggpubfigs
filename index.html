ggpubfigs
  • Quick Start

ggpubfigs is a ggplot2 extension that provides publication-ready themes and colorblind-friendly color palettes for the life sciences.

If you found ggpubfigs useful, please cite ggpubfigs: Colorblind-Friendly Color Palettes and ggplot2 Graphic System Extensions for Publication-Quality Scientific Figures. Steenwyk & Rokas. 2021, Microbiology Resource Announcements. doi: 10.1128/MRA.00871-21.


Quick StartÔÉÅ


Install

# install
devtools::install_github("JLSteenwyk/ggpubfigs")
# load
library(ggpubfigs)

Example 1: ito_seven + theme_big_simple

ggplot(mtcars, aes(factor(carb), fill=factor(cyl))) +
  geom_bar() +
  scale_fill_manual(values = friendly_pal("ito_seven")) +
  theme_big_simple()

Example 2: bright_seven + theme_big_grid

ggplot(mtcars, aes(factor(carb), fill=factor(cyl))) +
  geom_bar() +
  scale_fill_manual(values = friendly_pal("bright_seven")) +
  theme_big_grid()
