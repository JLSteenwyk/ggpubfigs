Color Palettes

Color palettes are all colorblind friendly. Thus, your figures will be accessible to more people. Color palettes are named using the following scheme: identifier (underscore) number of colors in color palette.


Discrete usage example

ggplot(mtcars, aes(factor(carb), fill=factor(cyl))) +
  geom_bar() +
  scale_fill_manual(values = friendly_pal("contrast_three"))

Continuous usage example

pal <- friendly_pal("contrast_three", 50, type = "continuous")
image(volcano, col = pal)
